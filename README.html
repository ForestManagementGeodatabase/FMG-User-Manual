1 FMG-User-Manual

The Forest Management Geodatabase (FMG) User Manual guides users through the processing of forestry field surveys into a geodatabase and producing summary reports.

1.1 Project Status

1.2 Description

This user manual guides users through using the FMG-toolbox to process FMG forestry field surveys into a geodatabase and produce summary reports. The FMG toolbox was developed by USACE foresters and geographers to implement the forest monitoring protocols established by the Upper Mississippi River Systemic Forest Stewardship Plan, Executive Summary.

1.3 Funding

Funding for development and maintenance of the FMG-toolbox has been provided by the following US Army Corps of Engineers (USACE) programs:

  • St. Louis District, Rivers Project Office
  • Rock Island District, Mississippi River Project
  • St. Paul District, Mississippi River Project
