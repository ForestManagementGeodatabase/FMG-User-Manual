1 Introduction

The FMG toolbox was developed by USACE foresters and geographers to implement the forest monitoring protocols established by the Upper Mississippi River Systemic Forest Stewardship Plan, Executive Summary.

The FMG is a geodatabase intended for use in Esri ArcGIS. The database is contained as an Esri personal geodatabase (.gdb), and should only be viewed, edited, etc. in ArcPro,ArcCatalog, ArcMap, or other Esri ArcGIS products.
